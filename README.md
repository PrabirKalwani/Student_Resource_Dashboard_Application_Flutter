## Deploy

To run this project create an .env file in the root of the project with the follwoing api keys :- 

```bash
  FIREBASE_WEB = <Your API Key>
  FIREBASE_ANDROID = <Your API Key>
  FIREBASE_IOS = <Your API Key>
  FIREBASE_MACOS = <Your API Key>

```
